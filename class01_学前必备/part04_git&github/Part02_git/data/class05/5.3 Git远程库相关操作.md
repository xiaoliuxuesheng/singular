# 5.3 Git远程库相关操作

### 1. Git远程版本库

- [Github](https://github.com/)
- [码云](https://gitee.com)

### 2.  push 和 pull的完整格式

```sh
git pull 远程URL 分支名称
```

```sh
git push 远程URL 分支名称
```

### 3. 本地库中配置远程库

- 配置连接远程库的URL

  ```sh
  git remote add 别名 URL
  ```

  > 连接远程库的操作使用别名即可

- 查看本地库中远程库信息

  ```sh
  git remote				 # 查看所有的远程仓储名称
  git remote show 别名		# 查看远程库别名对应的详细信息
  git remote  -v			 # 查看关联的所有的远程仓储名称及地址
  ```

- 删除远程库信息

  ```sh
  git remote remove 远程库别名
  ```
