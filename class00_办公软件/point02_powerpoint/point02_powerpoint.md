# 第二章 字体

1. 字体的选择
2. 字体样式设置：加粗、颜色、大小
3. **其他的字体特性都不要用**
4. 文字设计的相关技巧
   - **为文字添加外边框**：使字体外观上看着更整齐。用矩形形状添加边框并且去掉背景填充即可
   - **图文结合**：在图片上添加特定格式的字体，使图片更有冲击力，并且文字不会显得苍白
   - **镂空字**：主要用于突出文字：①在图片上添加一个矩形背景设置为黑色并添加透明度②添加文字③选择文字和半透明背景设置格式选项为减除 ⑤ 或者使用文字填充
   - **多文字的PPT**：①分段②商务字体③分段关键字使用商业蓝色作为标题④尽量删除文字
     - 优化①：为标题添加图标，只保留图标
     - 优化②：图片加段落布局
     - 优化③：大图标和部分文字
     - 优化④：纯文字使定义在一个框中，用logo辅助
     - 优化⑤：背景半透明框架加文字
5. **将文字转为形状**
