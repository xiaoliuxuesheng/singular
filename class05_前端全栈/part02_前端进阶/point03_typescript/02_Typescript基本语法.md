# 第二章 TypeScript数据类型

## 2.1 定义变量的格式

- 定义变量

    ```js
    let 变量名称:数据类型 = 初始值;
    ```

- 类型校验

    - 变量的数据类型初始化完成后,Typescript是不允许将其他类型的值赋值给改变量

## 2.2 数据类型

### 1. 布尔 : boolean

### 2. 数字 : number

### 3. 字符串 : string

### 4. 数组 : []

- 格式一

    ```js
    let 数组名称:数组元素类型[] = [];
    ```

- 格式二

    ```js
    let 数组名称:Array<数组元素类型> = [];
    ```

### 5. 元祖

> 属于数组的一种 : 可以包含多中数据类型

```js
let 元祖名称:[类型1,类型2] = [类型1的值,类型2的值];
```

- 为每个类型都定义数据类型

### 6. 枚举

- 定义枚举 : 枚举关键字 : **enum**

    ```js
    enum 枚举名称{
        枚举值1 = 初始化值,
        枚举值2,
        ... ...
    }
    ```

- 枚举值的初始化

    - 如果定义的枚举不初始化 : 默认值为枚举值对应的索引值
    - 如果定义枚举并初始化 : 枚举的值则为初始化值

### 7. 任意类型 : any

- 任意类型类型JavaScript中的变量的数据类型 : 根据值进行推断
- 任意类型在Typescript中不会进行类型校验

### 8. null 和 undefined

- 定义变量不赋值表示该对象为undefined类型 : 需要使用undefined关键字约束;

    ```js
    let 变量:undefined;
    ```

- 定义undefined类型和其他类型

    ```js
    let 变量:undefined | 其他类型;
    ```

- null类型的变量只可以赋值为`null`

### 9. void 类型

- 表示空 : 在定义方法时候如果方法没有返回值需要使用`void`声明 : 表示该方法没有返回值

### 10.其他类型 : never

- never **表示从来不会出现的值** : 匿名只执行函数将没有值

- `never`类型包括了`null` 和 `undefined`
- 声明`never` 类型的变量只能赋予`never`类型的值 : 否则编译报错

# 第三章 TypeScript 函数

## 3.1 函数的定义

- 定义函数格式一 : 声明式函数

    ```js
    function 函数名称():函数返回类型{
        //TODU
    }
    ```

- 定义函数格式二 : 匿名式函数

    ```js
    let 函数名 = function ():函数类型{
        //TODU
    }
    ```

## 3.2 函数的参数

### 1. 普通参数

```js
function (参数名称:参数类型,参数名称:参数类型){
    //TUDU
}
```

- 默认方法的形式参数和实际参数必须相同并且类型合法

### 2. 可选参数 : ?

```js
function (参数名称:参数类型,参数名称?:参数类型){
    //TUDU
}
```

- 在可选参数名称后面 添加符号 : `?`
- 设置可选参数 : 形式参数和实际参数可以不同

### 3. 默认参数

```js
function (参数名称:参数类型,参数名称:参数类型 = 默认值){
    //TUDU
}
```

- 默认参数必须是参数的最后

### 4. 可变参数 : `...`

```js
function (参数名称:参数类型, ...参数名称:参数类型[]){
    //TUDU
}
```

- 将多余的可变参数都会收集在数组中

## 3.3 函数的重载

- 通过为同一个函数提供多个函数类型定义 来实现多种功能的目的

- 定义重载方法

    ```js
    function 方法名称(参数名:参数类型):返回类型;
    function 方法名称():返回类型{
    	//TUDO : 实现两个方法的的逻辑
    }
    ```

    - 它查找重载列表，尝试使用第一个重载定义。 如果匹配的话就使用这个
    - 在定义重载的时候，**一定要把最精确的定义放在最前面**
