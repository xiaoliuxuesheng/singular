002_JVM与java上层结构

1. JVM与Java体系结构

   1. 前言

      - 首要要知道为什么学JVM：①中高级Java必备②不懂项目调优参数
      - 其次学习JVM的前提：①计算机组成原理②计算机网络③操作系统④数据结构⑤Java基础语法
      - 知道Java应用程序与JVM的关系：JVM-》javaAPI-》框架-》Java应用
      - 编程语言的分类：①机器指令②汇编语言③高级语言
      - 架构师的任务：①让系统更快②避免系统瓶颈
      - 为什么要学习JVM：①面试②项目管理调优③底层基础：垃圾回收算法、JIT、底层算法

   2. 面向人群与参考书

      1. 面向人群：①java开发②运维③架构师④技术爱好者
      2. 学习环境：JDK6~JDK13
      3. 参考书：①JDK规范7/8/11②Java虚拟机规范（翻译版）③深入理解Java虚拟机

   3. Java与JVM

      1. Java语言是生态丰富的语言：①作为一个平台②作为一种文化开源的代名词，以及丰富的开源第三方框架二③丰富的社区
      2. 跨平台语言：一次编译，到处运行
      3. Java虚拟机（JVM）：*网上搜一下JVM的介绍*
      4. 编译器： 将Java高级语言翻译为JVM字节码的工具
      5. JVM字节码：运行在JVM平台上的字节码格式都是一样的
      6. IT三大技术难题：①CUP②操作系统③编译器
      7. 多语言混合编程：因为不同邻域特定语言可以编译为Java字节码，在一个项目中可以使用多种语言根据语言特性完成各自的功能
