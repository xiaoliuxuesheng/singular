002_数据结构与算法概述

1. 什么是数据结构
   - 是把数据安装一定的关系组织起来的集合，用来存储和组织数据；
2. 数据结构分类
   - 逻辑结构：是根据具体问题抽象出来的模型，按照集合中元素之间的关系分类
     1. 集合：元素之间属于同一个数组，即存储在连续的内存地址中；
     2. 线性：数据元素之间存储一对一关系；
     3. 树形：数据元素之间存在一对多的层次关系；
     4. 图形：数据元素之间存在多对多的关系
   - 物理结构：是指数据在计算机中的标识形式，也称为存储结构；
     1. 顺序存储：数据存放在计算机的磁盘中的存储单元是连续的；
     2. 链式存储：数据存放在计算机的磁盘中的存储单元是非连续的，存储结构引入的指针用于存放数据元素的内存地址；
3. 什么是算法
   - 根据在给定的数据结构得到希望的结果而使用不同的计算策略称为算法

004_算法分析

1. 算法分析：研究算法的目的是：①用最少的时间②占用最小的内存；
   - 时间复杂度分析：是关于算法时间耗费的分析；
   - 空间复杂度分析：是关于算法内存耗费的分析；
2. 时间复杂度分析：是指数据规模和算法执行次数的函数关系；
3. 大O计法：
