## 1.1 数组介绍

- Java数组（Array）：一种基本数据存储结构，是多个相同类型数据按一定顺序排列的集合；
- 数组对应在内存中一段连续空间，数组的长度一旦确定，就不能修改；
- 数组里的每个元素都有编号，编号从0开始，并且依次递增；
- 同一个数组中的元素必须是同一类数据类型；
