# :anchor: 正则表达式作用

- 正则用于文本的复杂处理：功能强大而且灵活；
- 正则用于描述一种规则，通过这个规则可以配一种特定模式的字符串；
- 正则表达式学习的重点是语法规则；

# :anchor: 正则表达式语法规则

## 1. 普通字符

- 指字母、数字、下划线、汉字以及没有特殊含义的标点符号都称为普通字符；
- 普通字符在正则表达式中只能匹配与之相同的字符；

## 2. 特殊字符

- 特殊字符表示用来书写正则表达式的字符和文本中特殊符号的字符；

  | 转移字符                                                     | 作用                   |
  | ------------------------------------------------------------ | ---------------------- |
  | \n                                                           | 匹配一个换行符         |
  | \r                                                           | 匹配一个回车符         |
  | \t                                                           | 匹配一个制表符         |
  | \\\\                                                         | 匹配`\`本身            |
  | `\^`     `\^`     `\(`     `\)`<br />`\?`     `\+`     `\{`     `\}`<br />`\*`     `\|`     `\[`     `\]` | 正则表达式中的特殊字符 |

## 3. 标准字符集合

- 标准字符集合用来表示字代表文本中字母、数组、空白符的字符集合

  | 字符集合 | 含义                                       |
  | -------- | ------------------------------------------ |
  | \d       | 匹配一个数字字符。等价于 [0-9]。           |
  | \s       | 匹配任何空白字符，包括空格、制表符、换页符 |
  | \w       | 匹配字母、数字、下划线                     |

  > 大写都是对应的反义字符集合；

## 4. 自定义字符集合

- 表示使用方括号`[]`中自定义的字符集合

  | 自定义中特殊字符 | 含义                                       |
  | ---------------- | ------------------------------------------ |
  | `^`              | 表示取反：`[^abc]` 表示除了abc以外的字符   |
  | `-`              | 表示范围：`[2-9]` 表示2到9中的任意一个数字 |

  - 自定义字符集集合中特殊字符会失去特殊含义；
  - 可以使用反斜杠`\` 将自定义中特殊字符转义为普通字符；

## 5. 量词

- 量词用于修饰字符集合：包含标准字符集合和自定义字符集合，用于表示字符集合出现的次数；

  | 量词表示法 | 含义                          |
  | ---------- | ----------------------------- |
  | `{n}`      | 只重复n次                     |
  | `{m,n}`    | 至少重复m次，最多重复n次      |
  | `{m,}`     | 至少重复m次                   |
  | ？         | 重复0次或1次：相当于`{0,1}`   |
  | +          | 表示最少重复1次：相当于`{1,}` |
  | *          | 表示重复任意次：相当于`{0,}`  |

## 6. 字符边界

- 用于描述字符集合的边界位置

  | 边界符号    | 含义                                                         |
  | ----------- | ------------------------------------------------------------ |
  | `^字符集合` | 表示字符集合开始的地方                                       |
  | `字符集合$` | 表示字符集合结束的地方                                       |
  | `\b`        | 匹配一个单词边界：该字符位置前面的字符和后面的字符不全是`\w` |

## 7. 选择符

- 选择符：`|`

- 选择符左右两边都是正则表达式：作用是匹配左边或者匹配右边都成立

## 8. 捕获组与非捕获组

- 捕获组：`()`
  - 正则表达式中圆括号也称为捕获组：圆括号中的表达式被看做一个整体；
  - 被捕获组匹配的字符串会保存在内存中，可以获取出匹配结果，但是会消耗内存；
  - 获取匹配结果，每一对括号都会分配一个编号，编号是根据左括号顺序字段编号；
  - 返回引用格式：`()\编号`，`\编号`等价于获取出的字符串
- 非捕获组：`(?:)`
  - 因为捕获组匹配字符会将匹配的字符保存在内存中，消耗资源，如果不得不使用组表达式时候有不希望保存捕获字符，可以使用非捕获组，`(?:正则表达式)`格式的组表达式；

## 9. 预搜索 | 零宽断言 | 环视

- 也称为子表达式匹配：用于判断某一段正则表达式的前后位置是否也匹配另一个正则表达式；

- 这个子表达式匹配的字符串是的匹配宽度是0：匹配的字符是不会被记入匹配到的字符中；

  | 表达式          | 说明                     |
  | --------------- | ------------------------ |
  | `(?=子表达式)`  | 父表达式的后面可以匹配   |
  | `(?!子表达式)`  | 父表达式的后面不可以匹配 |
  | `(?<=子表达式)` | 父表达式的前面可以匹配   |
  | `(?<!子表达式)` | 父表达式的前面不可以匹配 |
