# 第二章 简介

## 2.1 配置中心

1. Disconf：百度开源的配置中心，不再维护了
2. SpringCloudConfig：Spring提供的配置中心组件
3. Apollo：携程开源
4. Nacos：阿里开源的配置中，可以DNS和RPC的服务发现

## 2.2 Apollo概述

- 可以集中化管理应用的不同环境、不同集群的配置；修改配置后能够实时推送到应用端；并且具备规范的权限、流程治理等特性，适用于微服务配置管理场景；
- Apollo包含服务端和客户端两部分：服务端基于SpringBoot和SpringCloud开发，可以直接打包部署；Java客户端不依赖任何框架，能够运行与所有java运行时环境，并且对Spring和SpringBoot环境支持；
