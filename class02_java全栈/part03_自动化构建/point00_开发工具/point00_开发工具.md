### 2. 全屏展示模式编码

- **进入全屏展示菜单选择**：<kbd>Alt + v</kbd>打开View菜单  ---》Enter Presentation Mode
  - 弹出最近编辑的代码：<kbd>Ctrl + e</kbd>
- **退出全屏展示菜单选择**：<kbd>Alt + v</kbd>打开View菜单  ---》ExitPresentation Mode
