### 5. 列表 list

 <font color=red size=5>:anchor: 元组定义：`[]`</font>

```python
列表名 = [元组元素, ... ...]
```

- 列表中每个元素会有一个数字表示他在列表中的位置，如果数字大于等于0表示需要从左向右读取，起始索引为0；如果数字小于0表示需要从右向左读取，起始索引为-1

 <font color=red size=5>:anchor: 内置函数</font>

| 函数名     | 说明                   |
| ---------- | ---------------------- |
| len(list)  | 计算列表元素个数。     |
| max (list) | 返回列表中元素最大值。 |
| min(list)  | 返回列表中元素最小值   |
| del list   | 删除列表               |

 <font color=red size=5>:anchor: 列表运算符</font>

| 运算符      | 说明 | 案例                                     | 结果                         |
| ----------- | ---- | ---------------------------------------- | ---------------------------- |
| +           | 连接 | [1, 2, 3] + [4, 5, 6]                    | [1, 2, 3, 4, 5, 6]           |
| *           | 复制 | ['Hi!']* 4                               | ['Hi!', 'Hi!', 'Hi!', 'Hi!'] |
| in          | 判断 | 3 in [1, 2, 3]                           | True                         |
| fot - in    | 迭代 | for x in [1, 2, 3]: <br />    print (x,) | 1 2 3                        |
| [索引]      | 读取 | [1, 2, 3\][2]                            | 2                            |
| [起始:终止] | 截取 | [1, 2, 3, 4, 5\][1:3]                    | 2,3                          |

- 列表索引理论上是有两个索引：顺序读取时候的索引和反向读取时候的索引；
- 如果根据索引读取，索引小于0则读取顺序是反向读取：顺序起始索引是0，反向起始索引是-1；
- 截取元组**不包含终止索引**：如果起始索引为空默认是起始索引0，如果终止索引为空默认是元组长度；

### 6. 字典 dict

<font color=red size=5>:anchor: 字典定义：`{}`</font>

```python
字典名 = {key1:value2, key2:value2, ...}
```

- 字典是可变容器模型，可存储任意类型
- 字典中key是唯一的且不可变，可以是数值、字符串、元组类型的，字典的值可以是任意类型的

<font color=red size=5>:anchor: 字典方法：`{}`</font>

| 函数                          | 说明                                                         |
| ----------------------------- | ------------------------------------------------------------ |
| del 字典名称                  | 删除字典                                                     |
| len(字典名称)                 | 计算字典元素个数                                             |
| str(字典)                     | 将字典输出为字符串                                           |
| type(字典)                    | 返回类型                                                     |
| radiansdict.clear()           | 删除字典内所有元素                                           |
| copy()                        | 返回一个字典的浅复制                                         |
| fromkeys()                    | 创建一个新字典，以序列seq中元素做字典的键，val为字典所有键对应的初始值 |
| get(key, default=None)        | 返回指定键的值，如果值不在字典中返回default值                |
| key in dict                   | 如果键在字典dict里返回true，否则返回false                    |
| items()                       | 以列表返回可遍历的(键, 值) 元组数组                          |
| keys()                        | 返回一个迭代器，可以使用 list() 来转换为列表                 |
| values()                      | 返回一个迭代器，可以使用 list() 来转换为列表                 |
| setdefault(key, default=None) | 和get()类似, 但如果键不存在于字典中，将会添加键并将值设为default |
| update(dict2)                 | 把字典dict2的键/值对更新到dict里                             |
| pop(key[,default])            | 删除字典给定键 key 所对应的值，返回值为被删除的值。key值必须给出。 否则，返回default值。 |
| popitem()                     | 随机返回并删除字典中的最后一对键和值。                       |

### 7. 元组 tuple

 <font color=red size=5>:anchor: 元组定义：`()`</font>

```python
元组名 = (元组元素, ... ...)
```

- 元组与数组类型，不同之处在于元组中的元素不可修改
- 如果元组中只有一个元素，需要在元素后添加一个逗号`，`，否则元组符号`()`会被当做括号运算符

 <font color=red size=5>:anchor: 内置函数</font>

| 函数名      | 说明                   |
| ----------- | ---------------------- |
| len(tuple)  | 计算元组元素个数。     |
| max (tuple) | 返回元组中元素最大值。 |
| min(tuple)  | 返回元组中元素最小值   |
| tuple(seq)  | 将列表转换为元组。     |
| del tuple   | 删除元组               |

 <font color=red size=5>:anchor: 元组运算符</font>

| 运算符      | 说明 | 案例                                     | 结果                         |
| ----------- | ---- | ---------------------------------------- | ---------------------------- |
| +           | 连接 | (1, 2, 3) + (4, 5, 6)                    | (1, 2, 3, 4, 5, 6)           |
| *           | 复制 | ('Hi!',) * 4                             | ('Hi!', 'Hi!', 'Hi!', 'Hi!') |
| in          | 判断 | 3 in (1, 2, 3)                           | True                         |
| fot - in    | 迭代 | for x in (1, 2, 3): <br />    print (x,) | 1 2 3                        |
| [索引]      | 读取 | (1, 2, 3)[2]                             | 2                            |
| [起始:终止] | 截取 | (1, 2, 3, 4, 5)[1:3]                     | 2,3                          |

- 元组的索引理论上是有两个索引：顺序读取时候的索引和反向读取时候的索引；
- 如果根据索引读取，索引小于0则读取顺序是反向读取：顺序起始索引是0，反向起始索引是-1；
- 截取元组**不包含终止索引**：如果起始索引为空默认是起始索引0，如果终止索引为空默认是元组长度；

### 8. 集合 set

 <font color=red size=5>:anchor: 集合定义</font>

```python
# 格式一
集合名 = {集合元素, ...}

# 格式二
集合名词 = set()
```

- 集合（set）是一个无序的不重复元素序列。
- 创建一个空集合必须用 set() 而不是 { }，因为 { } 是用来创建一个空字典
- 集合会把传入的字符串拆分为单个字符作为集合的成员，当传入列表时，集合会把列表内的元素作为集合成员

| clear()                       | 移除集合中的所有元素                                         |
| copy()                        | 拷贝一个集合                                                 |
| difference()                  | 返回多个集合的差集                                           |
| difference_update()           | 移除集合中的元素，该元素在指定的集合也存在。                 |
| discard()                     | 删除集合中指定的元素                                         |
| intersection()                | 返回集合的交集                                               |
| intersection_update()         | 返回集合的交集。                                             |
| isdisjoint()                  | 判断两个集合是否包含相同的元素，如果没有返回 True，否则返回 False。 |
| issubset()                    | 判断指定集合是否为该方法参数集合的子集。                     |

## 3.4 Python定义变量

```python
变量名称 = 变量指
```

- **变量类型 : ** 变量存储在内存中的值。这就意味着在创建变量时会在内存中开辟一个空间。基于变量的数据类型，解释器会分配指定内存，并决定什么数据可以被存储在内存中。因此，变量可以指定不同的数据类型，这些变量可以存储整数，小数或字符。
- **变量赋值 : **
  - 每个变量在内存中创建，都包括变量的标识，名称和数据这些信息。
  - 每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。
  - 等号（=）用来给变量赋值 : 等号运算符左边是一个变量名,等号运算符右边是存储在变量中的值

## 3.6 Python基本语法

### 1. 顺序结构

- 从上向下执行

### 2. 分值结构

- if

  ```python
  if 逻辑表达式:
      pass
  ```

- if - else

  ```python
  if 逻辑表达式:
      pass
  else:
      pass
  ```

- if - elif

  ```python
  if 逻辑表达式1:
      pass
  elif 逻辑表达式2:
      pass
  ```

- if - elif - else

  ```python
  if 逻辑表达式1:
      pass
  elif 逻辑表达式2:
      pass
  else:
      pass
  ```

### 3. 循环结构

- while

  ```sh
  while 逻辑表达式:
      pass
  ```

- for - in

  ```python
  for item in 集合:
      pass
  ```

★ 退出循环

- break : 某一条件满足的时候，退出循环，不再执行后续重复的代码
- continue : 某一条件满足的时候，不执行本次循环的后续代码，继续执行下次循环
- exit() : 直接退出程序

### 4. 迭代器与可迭代对象

- 可迭代对象
  - 可以直接作用于for循环的对象统称为可迭代对象
- 迭代器
  - 迭代器是一个可以记住遍历的位置的对象；
  - 迭代器对象从集合的第一个元素开始访问，直到所有的元素被访问完结束。迭代器只能往前不会后退；
  - 迭代器有两个基本的方法：**iter()** 和 **next()**；
  - 字符串，列表或元组对象都可用于创建迭代器。
