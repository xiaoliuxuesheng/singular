### 5. 列表 list

 <font color=red size=5>:anchor: 元组定义：`[]`</font>

```python
列表名 = [元组元素, ... ...]
```

- 列表中每个元素会有一个数字表示他在列表中的位置，如果数字大于等于0表示需要从左向右读取，起始索引为0；如果数字小于0表示需要从右向左读取，起始索引为-1

 <font color=red size=5>:anchor: 内置函数</font>

| 函数名     | 说明                   |
| ---------- | ---------------------- |
| len(list)  | 计算列表元素个数。     |
| max (list) | 返回列表中元素最大值。 |
| min(list)  | 返回列表中元素最小值   |
| del list   | 删除列表               |

 <font color=red size=5>:anchor: 列表运算符</font>

| 运算符      | 说明 | 案例                                     | 结果                         |
| ----------- | ---- | ---------------------------------------- | ---------------------------- |
| +           | 连接 | [1, 2, 3] + [4, 5, 6]                    | [1, 2, 3, 4, 5, 6]           |
| *           | 复制 | ['Hi!']* 4                               | ['Hi!', 'Hi!', 'Hi!', 'Hi!'] |
| in          | 判断 | 3 in [1, 2, 3]                           | True                         |
| fot - in    | 迭代 | for x in [1, 2, 3]: <br />    print (x,) | 1 2 3                        |
| [索引]      | 读取 | [1, 2, 3\][2]                            | 2                            |
| [起始:终止] | 截取 | [1, 2, 3, 4, 5\][1:3]                    | 2,3                          |

- 列表索引理论上是有两个索引：顺序读取时候的索引和反向读取时候的索引；
- 如果根据索引读取，索引小于0则读取顺序是反向读取：顺序起始索引是0，反向起始索引是-1；
- 截取元组**不包含终止索引**：如果起始索引为空默认是起始索引0，如果终止索引为空默认是元组长度；

## 3.4 Python定义变量

```python
变量名称 = 变量指
```

- **变量类型 : ** 变量存储在内存中的值。这就意味着在创建变量时会在内存中开辟一个空间。基于变量的数据类型，解释器会分配指定内存，并决定什么数据可以被存储在内存中。因此，变量可以指定不同的数据类型，这些变量可以存储整数，小数或字符。
- **变量赋值 : **
  - 每个变量在内存中创建，都包括变量的标识，名称和数据这些信息。
  - 每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。
  - 等号（=）用来给变量赋值 : 等号运算符左边是一个变量名,等号运算符右边是存储在变量中的值

  ```python
  if 逻辑表达式1:
      pass
  elif 逻辑表达式2:
      pass
  else:
      pass
  ```

### 3. 循环结构

- while

  ```sh
  while 逻辑表达式:
      pass
  ```

- for - in

  ```python
  for item in 集合:
      pass
  ```

★ 退出循环

- break : 某一条件满足的时候，退出循环，不再执行后续重复的代码
- continue : 某一条件满足的时候，不执行本次循环的后续代码，继续执行下次循环
- exit() : 直接退出程序

### 4. 迭代器与可迭代对象

  - 可以直接作用于for循环的对象统称为可迭代对象

  - 迭代器是一个可以记住遍历的位置的对象；

# 第四章 模块

## 4.1 包与模块的介绍

:anchor: <font size=4 color=blue>Python项目组织</font>

- **包**：本质是一个文件夹，包的特征是包含一个`__init__.py`文件；
- **模块**：包中的py文件称为模块，其他的称为Python脚本文件；
  - **类**：定义在模块中的代码结构，一个模块可以包含多个类
  - **函数**：函数式编程中直接定义在模块中的函数，定义在类中的函数称为方法；
  - **变量**：定义在模块中的变量，包含全局变量和局部变量

:anchor: <font size=4 color=blue>包与模块的名称</font>

- **包的名称**：是文件夹的名称；
- **模块的名称**：就是Python文件的名称
  - `__init__.py`也是一个特殊的模块，这个模块的名称是他所在的包的名称；
- **模块命名空间**：`包名.模块`

## 4.2 模块的导入

:anchor: <font size=4 color=blue>import格式</font>

1. 导入单个模块

   ```python
   # 导入
   import 模块命名空间
   
   # 使用模块中成药
   变量 = 模块命名空间.模块成员
   ```

   > 这种导入方式命名空间过长导致代码复杂

2. 为导入模块定义别名

   ```python
   # 导入
   import 模块命名空间 as 别名
   
   # 使用模块中成员
   变量 = 别名.模块成员
   ```

:anchor: <font size=4 color=blue>`from ··· import`格式</font>

1. 直接引入模块中的成员

   ```python
   # 导入
   from 模块命名空间 import  模块成员
   
   # 使用模块中成员
   变量 = 模块成员
   ```

2. 使用通配符代表模块成员

   ```python
   # 导入
   from 模块命名空间 import  *
   
   # 使用模块中成员
   变量 = 模块成员
   ```

   - 使用模块内置属性：`__all__=[成员1,成员2, ... ]`，在模块中定义这个属性，则其他模块只可以引入这个模块的指定变量；

3. 一次引入单个模块的多个成员

   ```python
   # 导入
   from 模块命名空间 import (成员1,成员2, ...)
   
   # 使用模块中成员
   变量 = 成员1
   ```

4. 使用`__init__.py`引入多个模块

   - 当包或包中的模块被引入时，`__init__.py`这个文件会被自动执行；
   - 可以根据`__init__.py`模块的特性，在当前包中模块的引入项都定义在`__init__.py`这个模块中；
   - 在`__init__.py`中使用模块内置属性`__all__=[成员1,成员2, ... ]`，可以规定当前保证指定可以被引用的模块；

   ```python
   import sys
   import file
   
   __all__ = [模块1, 模块2, ...]
   ```

## 4.3 模块导入说明

- 相同的模块不会被重复引入；
- 在模块引入时应该避免循环引入；
- 引入其他模块就是执行被引入模块中的代码：一个应用程序中一般只会设置一个入口文件；

# 第五章 函数

## 5.1 函数的定义与调用

:anchor: <font size=4 color=blue>定义函数</font>

```python
def 函数名称([形参1, 形参2, ...]): 
    pass
```

:anchor: <font size=4 color=blue>函数调用</font>

```python
函数名称([实参1, 实参2, ...])
```

## 5.2 函数的类型

:anchor: <font size=4 color=blue>无参无返回</font>

```python
def 函数名称():
    pass
```

:anchor: <font size=4 color=blue>无参有返回</font>

```python
def 函数名称():
    return 函数返回值
```

:anchor: <font size=4 color=blue>有参无返回</font>

```python
def 函数名称([形参1, 形参2, ...]):
    pass
```

:anchor: <font size=4 color=blue>有参有返回</font>

```python
def 函数名称([形参1, 形参2, ...]):
    return 函数返回值
```

## 5.3 函数参数

:anchor: <font size=4 color=blue>必须参数</font>

- 函数定义

  ```python
  def 函数名称([形参1, 形参2, ...]):
      pass
  ```

- 函数调用

  ```python
  函数名称([实参1, 实参2, ...])
  ```

:anchor: <font size=4 color=blue>关键字参数</font>

- 函数定义

  ```python
  def 函数名称([形参1, 形参2, ...]):
      pass
  ```

- 关键字给函数赋值

  ```python
  函数名称(形参名2=值, 形参1=值)
  ```

:anchor: <font size=4 color=blue>默认参数</font>

- 定义函数指定参数默认值

  ```python
  def 函数名称([形参1, 形参2=默认值]):
      pass
  ```

- 函数调用：默认值参数如果不传就使用默认值；

:anchor: <font size=4 color=blue>元组可变参数</font>

- 将可变参数定义在函数中

  ```python
  def 函数名称([形参1, 形参2, ..., *可变参数名称]):
      pass
  ```

  - 参数前缀使用单`*`表示该参数是元组可变参数；
  - 元组可变参数用户接受实际参数中超过形式参数的其他参数；
  - 元组可变参数必须定义在形式参数列表的后面;

:anchor: <font size=4 color=blue>字典可变参数</font>

- 字典可变参数定义在函数中

  ```python
  def 函数名称([形参1, 形参2, ..., *可变参数名称, **字典可变参数]):
      pass
  ```

  - 字典可变参数必须定义在所有参数的最后面

- 字典可变参数的调用

  ```python
  函数名称(实际参数,key1=value1, key2=value2)
  ```

  - 字典可变参数用于接收实际参数中多余的`key=value`格式的参数

## 5.4 全局变量和局部变量

- 全局变量：表示定义在函数外部的变量表示全局变量

- 局部变量：表示定义在函数内部的变量

  - 如果局部变量名与全局变量同名：在函数内容默认使用的是局部变量

  - 如果局部变量名与全局变量同名并且需要在函数内部使用全局变量：需要显示声明使用的是全局变量

    ```python
    variable = 值;
    def函数名称():
        global variable
    ```

## 5.5 内部函数

- 内部函数：表示定义在函数内部的函数；

  ```python
  def 函数名称():
      变量 = 值
      ...
      def 内部函数():
          nonlocal 变量
          pass
      
      内部函数名()
  ```

  - 内部函数使用外部函数的变量需要加关键字：nonlocal关键字
  - 在函数内部使用内置函数：locals（）可以查看函数内部的本地变量成员，以字典的形式输出；
  - 在模块内部时候内置函数：globals（）可以查看模块内容的全局成员，一字典的形式输出；

## 5.6 闭包

:anchor: <font size=4 color=blue>闭包的条件</font>

1. 在外部函数中定义了内部函数；
2. 外部是由返回值的；
3. 外部的函数的返回值是内部函数；
4. 内部函数引用的外部函数中的变量；

:anchor: <font size=4 color=blue>格式</font>

```python
def 函数名称():
    外部变量 = 值
    ...
    def 内部函数():
        nonlocal 外部变量
        pass
    
    return 内部函数
```
